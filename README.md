Ocean View Resort Management System

This project is a web based hotel reservation management system developed using Java EE technologies. The system allows resort staff to manage guest reservations, calculate bills, and handle booking information easily.

Features
- User Login Authentication
- Add New Reservation
- View Reservations
- Edit Reservation
- Delete Reservation
- Automatic Bill Calculation
- Help Section

Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Bootstrap
- JUnit
- Git & GitHub

System Architecture
The system follows a three tier architecture:

- Presentation Layer– JSP pages for user interface  
- Business Logic Layer– Service classes handling system logic  
- Data Access Layer– DAO classes managing database operations  

Local Setup
Follow these steps to run the project locally:

1. Clone the repository from GitHub  
  -https://github.com/Fahad-042/OceanViewResort-System

2. Open the project in IntelliJ IDEA or Eclipse  
3. Configure Apache Tomcat server  
4. Set up the MySQL database and import the project database  
5. Update database connection settings in the DBConnection file  
6. Run the project on the server  
7. Open the system in your browser
